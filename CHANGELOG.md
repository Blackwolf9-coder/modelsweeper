# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-16

### Added
- Desktop app foundation with Electron + React + TypeScript.
- Role-based preset system (`Thinker`, `Coder`, `Reviewer`, custom slots).
- Model catalog with search, sorting, filtering, and direct role assignment.
- Hardware Calculator for VRAM/RAM/on-disk planning.
- Integration payload generation and sync support.
- Project governance:
  - Contributing guide
  - Security policy
  - Code of conduct
  - Issue and PR templates
  - CI workflow

### Improved
- Generic product branding under `ModelSweeper`.
- README structure, badges, and quick links.
- Launch plan and landing page content for public rollout.
