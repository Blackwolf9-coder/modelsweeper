## ModelSweeper v0.1.0

First public release of ModelSweeper.

### What ships in this version
- Desktop control center for AI model workflows
- Role-based presets:
  - Thinker
  - Coder
  - Reviewer
  - Custom slots
- Model inventory and role assignment from one view
- Hardware Calculator (VRAM/RAM/unified memory fit)
- Integration payload sync for toolchain consistency

### Production hardening
- CI workflow for typecheck/lint/build
- Issue templates and PR template
- Contributing, security, and code of conduct policies
- MIT license and changelog
