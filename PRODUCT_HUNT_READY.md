## Full Description
ModelSweeper is a desktop command center for local and hybrid AI workflows.

It helps you:
- switch between role-based model presets (Thinker, Coder, Reviewer)
- check RAM/VRAM feasibility before runtime
- organize model decisions and integration settings in one place

The goal is simple: less setup friction, fewer memory surprises, and faster execution.
